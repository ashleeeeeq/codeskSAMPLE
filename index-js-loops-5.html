Loops

Loops in JavaScript are used to repeatedly execute a block of code until a particular condition is met. They provide a way to automate repetitive tasks and iterate over arrays or other data collections.


Loops provide powerful control flow mechanisms in JavaScript, allowing you to repeat code execution until a condition is met. They are essential for tasks that involve iterating over collections, processing data, and implementing algorithms.


1. For Loop: The for loop is commonly used when you know the number of iterations beforehand. It consists of an initialization, a condition, and an update expression.


Example:


for (var i = 0; i < 5; i++) {
    console.log(i);
}


In this example, the for loop will iterate five times, printing the value of i from 0 to 4.


2. While Loop: The while loop executes a code block if a specified condition is true.


Example:


var i = 0;
while (i < 5) {
    console.log(i);
    i++;
}


In this example, the while loop will execute the code block if i is less than 5, printing the value of i from 0 to 4.


3. Do-While Loop: The do-while loop is similar to the while loop but guarantees that the code block executes at least once before checking the condition.


Example:


var i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);


In this example, the code block will execute once, printing the value of i as 0, even though the condition is initially false.


4. For...in Loop: The for...in the loop is used to iterate over the properties of an object.


Example:


const person = {
    name: "John",
    age: 30,
    city: "New York"
};

for (var key in person) {
    console.log(key + ": " + person[key]);
}


In this example, the for...in loop iterates over the properties of the person object, printing the key-value pairs.


5. For...of Loop: The for...of the loop is used to iterate over the elements of an iterable object, such as an array or a string.


Example:


const fruits = ["apple", "banana", "orange"];

for (var fruit of fruits) {
    console.log(fruit);
}


In this example, the for...of loop iterates over the fruits array, printing each element.
