Methods

In C#, methods are used to encapsulate a set of instructions that perform a specific task or return a value. Here is an example of a basic method in C#:


public int Add (int a, int b)
{
    return a + b;
}


Let's break down the syntax:


  • ‘public’: This is an access modifier that determines the method’s visibility. In this case, the method is public, meaning it can be accessed from any program part.
  • ‘int’: This is the method’s return type, which specifies the type of value that the method returns. In this case, the method returns an integer.
  • ‘Add’: This is the name of the method.
  • ‘(int a, int b)’: These are the method’s parameters. The method takes two integer parameters, named a and b.
  • ‘return a + b;’: The method’s body performs the task and returns a value. In this case, the method adds the two parameters, a and b, and returns the result.

To use this method, you would call it from another part of the program, passing in two integer values:


int result = Add (3, 5);


This would set ‘result’ to the value 8.
