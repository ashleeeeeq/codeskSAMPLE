Arrays

Arrays in JavaScript are used to store multiple values in a single variable. They are special object that holds an ordered collection of elements. Arrays can contain elements of different data types, such as numbers, strings, objects, or even other arrays. Each element in an array has an index starting from 0.


Arrays in JavaScript are versatile data structures that allow you to store, access, and manipulate collections of elements. They are commonly used in various programming tasks, such as iterating over elements, sorting data, or implementing data structures like stacks and queues.


1. Array Declaration: An array can be declared using square brackets [], separating each element with a comma.


Example:


var fruits = ["apple", "banana", "orange"];


In this example, an array fruits is declared and initialized with three elements: "apple", "banana", and "orange".


2. Accessing Array Elements: You can access individual elements of an array using their index within square brackets [].


Example:

var fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // Output: "apple"
console.log(fruits[1]); // Output: "banana"


In this example, the first element of the fruits array is accessed using the index 0, and the second element is accessed using the index 1.


3. Modifying Array Elements: You can modify array elements by assigning new values to specific indices.


Example:

var fruits = ["apple", "banana", "orange"];
fruits[1] = "grape";
console.log(fruits); // Output: ["apple", "grape", "orange"]


In this example, the element at index 1 of the fruits array is modified to "grape".


4. Array Length: The length property can be used to determine the number of elements in an array.


Example:


var fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // Output: 3


In this example, the length property of the fruits array returns 3, indicating the number of elements in the array.


5. Array Methods: JavaScript provides a variety of built-in methods to manipulate arrays, such as push(), pop(), splice(), and concat(). These methods allow you to add or remove elements, extract portions of an array, or combine arrays.


Example:

var fruits = ["apple", "banana", "orange"];
fruits.push("mango"); // Add an element to the end of the array
console.log(fruits); // Output: ["apple", "banana", "orange", "mango"]
fruits.pop(); // Remove the last element from the array
console.log(fruits); // Output: ["apple", "banana", "orange"]


In this example, the push() method adds "mango" to the end of the fruits array, and the pop() method removes the last element.
