Conditionals

Conditionals in JavaScript allow you to make decisions and execute different blocks of code based on certain conditions. Three types of conditional statements are commonly used: if, if-else, and switch.


1. If Statement: The if statement allows you to execute a code block if a specified condition is true. If the condition is false, the code block is skipped.


Example:


var age = 18;

if (age >= 18) {
    console.log("You are eligible to vote.");
}


In this example, if the age variable is 18 or greater, the message "You are eligible to vote" will be displayed.


2. If-Else Statement: The if-else statement provides an alternative execution path. If the condition specified in the if statement is true, the code block within the if statement is executed. The code block within the else statement is executed if the condition is false.


Example:

var temperature = 25;
if (temperature > 30) {
    console.log("It's a hot day!");
} else {
    console.log("It's a pleasant day.");
}
; }


In this example, if the temperature is greater than 30, the message "It's a hot day!" will be displayed. Otherwise, the message "It's a pleasant day." will be displayed.


3. Switch Statement: The switch statement provides a way to select one of many code blocks to be executed. It evaluates an expression and compares it with different cases. If a match is found, the corresponding code block is executed.


Example:


var day = "Monday"; switch (day) {
    case "Monday":
        console.log("It's the start of the week.");
        break;
    case "Friday":
        console.log("It's almost the weekend!");
        break;
    default:
        console.log("Enjoy your day!");
}


In this example, if the value of the day is "Monday", the message "It's the start of the week." will be displayed. If it's "Friday", the message "It's almost the weekend!" will be displayed. If none match, the default case will execute, and "Enjoy your day!" will be displayed.


These examples illustrate how conditionals in JavaScript allow you to control the flow of your program based on specific conditions. You can use logical operators, comparison operators, and various expressions to create more complex conditions within these statements.
