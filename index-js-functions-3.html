Functions

Functions in JavaScript are reusable blocks of code that can be called and executed when needed. They allow you to group related code, perform specific tasks, and organize your code into modular components. Functions can accept input parameters, perform actions, and return values.


Functions in JavaScript are powerful tools that enable code reuse, modularization, and abstraction. They can be invoked multiple times with different arguments, allowing for flexible and efficient coding. You can pass functions as arguments, store them in variables, and even define functions within functions.


1. Function Declaration: A function is declared using the function keyword followed by the function name, a list of parameters enclosed in parentheses, and a code block.


Example:


function greet(name) {
    console.log("Hello, " + name + "!");
}

greet("John");


In this example, the greet function is defined to take a name parameter. When the function is called with the argument "John", it will print "Hello, John!" to the console.


2. Function Expression: A function expression assigns a function to a variable. It can be anonymous or named.


Example:


var greet = function(name) {
    console.log("Hello, " + name + "!");
};

greet("Jane");

In this example, an anonymous function is assigned to the greet variable. It accepts a name parameter and prints a greeting to the console when called with an argument.


3. Arrow Function: Arrow functions provide a concise syntax for writing functions. They are beneficial for one-line functions or callbacks.


Example:


var multiply = (a, b) => a * b;

console.log(multiply(5, 3)); // Output: 15


In this example, an arrow function multiply takes two parameters, a and b, and returns their product. The result is then printed to the console.


4. Returning Values: Functions can return values using the return keyword. The returned value can be assigned to a variable or used directly.


Example:


function add(a, b) {
    return a + b;
}
var result = add(2, 3);
console.log(result); // Output: 5


In this example, the add function takes two parameters and returns their sum. The returned value is assigned to the result variable and then printed to the console.
