Object-Oriented Programming

How does Object-Oriented Programming work in Java?


Object-oriented programming is a programming paradigm that uses objects to represent data and behaviors. Java is an object-oriented programming language that supports the four basic principles of object-oriented programming: encapsulation, inheritance, abstraction, and polymorphism.


In Java, everything is an object except for the primitive data types (such as int, float, etc.). Objects in Java are instances of classes, which are templates for creating objects. A class defines the properties (fields) and behaviors (methods) of objects that belong to it.


To use an object in Java, you must first create an instance of the class that defines it. This is done using the "new" keyword followed by the name of the class and any arguments that the class constructor requires. For example:


Person john = new Person("John", 25);


This creates an instance of the Person class with the name "John" and age 25 and assigns it to the variable "john".


Once you have an instance of a class, you can access its fields and methods using dot notation. For example:


System.out.println(john.getName()); // prints "John"
john.setAge(26); // sets the age of the person to 26


This code accesses the "name" field and "getName()" and "setAge()" methods of the "john" object.
In Java, you can also create subclasses that inherit properties and behaviors from a parent class. This is done using the "extends" keyword. For example:


public class Student extends Person {
    private int studentId;

    public Student(String name, int age, int studentId) {
        super(name, age);
        this.studentId = studentId;
    }
    public int getStudentId() {
        return studentId;
    }
}


This code creates a subclass of the Person class called Student, which adds a "studentId" field and a "getStudentId()" method. The "super" keyword is used to call the constructor of the parent class.


Overall, Java's object-oriented programming features allow for creation of reusable and modular code, making it a popular choice for software development.
