Arrays and Loops

Arrays


Arrays in C# store a collection of similar types of elements. The elements of an array can be accessed using an index, which starts from 0. The general syntax for declaring an array in C# is as follows:


dataType[] arrayName = new dataType[arraySize];


For example, to declare an array of integers that can hold five (5) elements, we can use the following code:


int[] myArray = new int[5];


We can also initialize the elements of an array during its declaration as follows:


int[] myArray = { 10, 20, 30, 40, 50 };




Loops


In C#, loops are used to execute a code block repeatedly until a particular condition is met. There are three types of loops in C#: ‘for,’ ‘while,’ and ‘do-while’ loops.




The general syntax for a ‘for’ loop in C# is as follows:


for (initialization; condition; increment/decrement)
{
    // code block to be executed
}


For example, the following code prints the numbers from 1 to 5:


for (int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}




The general syntax for a ‘while’ loop in C# is as follows:


while (condition)
{
    // code block to be executed
}


For example, the following code prints the numbers from 1 to 5 using a ‘while’ loop:


int i = 1;
while (i <= 5)
{
    Console.WriteLine(i);
    i++;
}




The general syntax for a ‘do-while’ loop in C# is as follows:


do
{
    // code block to be executed
} while (condition);


For example, the following code prints the numbers from 1 to 5 using a ‘do-while’ loop:


int i = 1;
do
{
    Console.WriteLine(i);
    i++;
} while (i <= 5);
