Variables

In Java, variables store data that a program can use and manipulate.


To declare a variable in Java, you need to specify the data type of the variable followed by the variable name. For example:


int age; // declares a variable of type integer named "age"
double salary; // declares a variable of type double named "salary"
String name; // declares a variable of type String named "name"


You can also initialize a variable at the time of declaration like this:


int age = 30; // declares an integer variable named "age" and assigns it a value of 30.
double salary = 50000.0; // declares a double variable named "salary" and assigns it a value of 50000.0
String name = "John"; // declares a String variable named "name" and assigns it a value of "John"


Once a variable has been declared and initialized, you can use it in your code. For example:


System.out.println("My name is " + name + " and I am " + age + " years old.");


This code will print out a message that includes the value of the "name" and “age” variables.


Variables in Java can be of several different types, including primitive types (such as int, double, and boolean) and reference types (such as String, arrays, and objects). Each type has its declaration, initialization, and use rules, so it's important to understand their differences.
