Data Types and Variables

In C#, data types and variables store and manipulate data in a program. A data type is a classification of data that tells the compiler how to handle a piece of data. At the same time, a variable is a named memory location that holds a value of a specific data type.


Here are some common data types and variables in C#:


Data Types


  1. ‘bool’: Represents a Boolean value, which can be true or false.
  2. ‘int’: Represents a 32-bit signed integer, which can hold values between -2,147,483,648 and 2,147,483,647.
  3. ‘float’: Represents a single-precision floating-point number, which can hold values with up to 7 digits of precision.
  4. ‘double’: Represents a double-precision floating-point number, which can hold values with up to 15-16 digits of precision.
  5. ‘char’: Represents a single Unicode character enclosed in single quotes, e.g., 'A.’
  6. ‘string’: Represents a sequence of Unicode characters enclosed in double quotes, e.g., "Hello, World!".

Variables


To declare a variable in C#, specify its data type, followed by its name and, optionally, an initial value. Here are some examples:


int age; // declaring an integer variable named age
double price = 19.99; // declaring a double variable named price and assigning it an initial value of 19.99
string name = "John"; // declaring a string variable named name and assigning it an initial value of "John"


Once you have declared a variable, you can assign a value using the assignment operator ‘=.’ Here's an example:


age = 25; // assigning a value of 25 to the age variable


You can also declare and initialize a variable in a single statement like this:


int count = 0; // declaring and initializing an integer variable named count with an initial value of 0


To use a variable in your program, you can refer to its name. For example, to print the value of the ‘age’ variable to the console, you can use the ‘Console.WriteLine’ method like this:


Console.WriteLine(age); // prints the value of the age variable to the console
