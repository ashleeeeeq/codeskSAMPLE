Logic and Conditionals

Logical Operators


In C#, you can use logical operators and conditional statements to control the flow of your program. Logical operators allow you to combine conditions, while conditional statements enable you to execute different codes based on those conditions.

There are three logical operators in C#:


Here are some common data types and variables in C#:


  1. ‘&&’ (logical AND): Returns true if both operands are true.
  2. ‘||’ (logical OR): Returns true if either operand is true.
  3. ‘!’ (logical NOT): Returns the opposite of the operand's value (true if the operand is false, and vice versa).

Here's an example of using logical operators to combine conditions


int age = 25;
bool isStudent = true;
if (age >= 18 && !isStudent) {
    Console.WriteLine("You are eligible to vote.");
} else {
    Console.WriteLine("You are not eligible to vote.");
}


In this example, we use the ‘&&’ operator to combine two conditions: ‘age >= 18’ and ‘!isStudent’. The first condition checks if the ‘age’ variable is greater than or equal to 18. The second condition checks if the ‘isStudent’ variable is false (using the ‘!’ operator to negate its value). If both conditions are ‘true,’ the code inside the ‘if’ statement is executed, which prints "You are eligible to vote." Otherwise, the code inside the ‘else’ statement is executed, which prints "You are not eligible to vote."




Conditional Statements


There are two conditional statements in C#: ‘if’ and ‘switch.’

‘if’ statement


The ‘if’ statement allows you to execute different code based on a condition. Here's an example:


int age = 25;
if (age >= 18) {
    Console.WriteLine("You are eligible to vote.");
}


In this example, we check if the ‘age’ variable is greater than or equal to 18 using the ‘>=’ operator. If the condition is ‘true,’ the code inside the ‘if’ statement is executed, which prints "You are eligible to vote."

You can also use the ‘else’ keyword to specify what code to execute if the condition is ‘false’:


int age = 15;
if (age >= 18) {
    Console.WriteLine("You are eligible to vote.");
} else {
    Console.WriteLine("You are not eligible to vote.");
}


In this example, because the ‘age’ variable is less than 18, the condition is ‘false,’ so the code inside the ‘else’ statement is executed, which prints "You are not eligible to vote." ‘switch’ statement The ‘switch’ statement allows you to execute different codes based on the value of a variable. Here's an example:


int dayOfWeek = 3;
switch (dayOfWeek) {
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    case 4:
        Console.WriteLine("Thursday");
        break;
}


In this example, we use the ‘switch’ statement to print the name of the day of the week based on the value of the ‘dayOfWeek’ variable. The ‘case’ keyword specifies the value to match, and the code inside each case block is executed if the value matches. The ‘default’ keyword specifies what code to run if none of the values matches.
